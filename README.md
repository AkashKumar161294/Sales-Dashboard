# Sales-Dashboard
**🛒 Sales Dashboard – Supermarket Shop
****📌 Overview
**The Sales Dashboard provides a comprehensive view of supermarket performance by consolidating sales, profit, product, category, sales type, and payment mode metrics into a single, interactive interface.
It is designed to help business users and stakeholders quickly monitor trends, identify top performers, and make data‑driven decisions.
The dashboard uses a dark-themed layout with clearly separated KPI tiles and visual sections for easy interpretation.
